# How to set up

**Required Dependencies**

`discord.js`,
`@discordjs/builders`,
`@discordjs/rest`,
`discord-api-types`

**Configuration File**

In `config.json` replace your data with the placeholders

``{
  "token": "YOUR_BOT_TOKEN",``

  ``
  "clientId": "YOUR_CLIENT_ID",``

  ``
  "guildId": "YOUR_GUILD_ID",

  
  "roleId": "YOUR_ROLE_ID"
}``
